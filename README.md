# FarmConnect - Farmer to Buyer Marketplace

FarmConnect is a web application that connects farmers directly with buyers, eliminating middlemen and providing better prices for both parties.

## Features

### For Farmers:
- List products with real-time market price comparison
- Manage inventory and orders
- Direct chat with buyers
- Dashboard with earnings analytics
- Secure payment with 3% platform commission

### For Buyers:
- Browse fresh produce directly from farmers
- Real-time market price display
- Direct chat with farmers
- Secure ordering system
- Order tracking and management

## Tech Stack

### Backend:
- Node.js with Express.js
- MongoDB with Mongoose
- Socket.io for real-time chat
- JWT for authentication

### Frontend:
- HTML5, CSS3, JavaScript (Vanilla)
- Responsive design with Flexbox/Grid
- Socket.io client for real-time features
